Цей репозиторій містить вихідний код C++ для двоканального генератора прямого цифрового синтезу (DDS), розробленого спеціально для прецизійного просторового фокусування в системах **темпоральної інтерференції (Temporal Interference, TI)** магнітних полів.

## 🚀 Головна інновація: 12-ричний синтез (Base-12 DDS)

Стандартні системи генерації хвиль використовують двійкові акумулятори фази (32-bit) та обчислення з рухомою комою (float) через ірраціональне число $\pi$. Для двоканальних систем просторової інтерференції це створює критичну проблему: **фазовий джитер (jitter) та похибки округлення**, які руйнують фокус поля у просторі.

Цей проєкт вирішує проблему на фундаментальному математичному рівні, використовуючи **12-ричну цілочисельну матрицю**. 

* **Матриця 1728:** Повний цикл хвилі (360°) розбито на $12^3 = 1728$ кроків. Число 12 ідеально ділиться на 2, 3, 4 та 6, що забезпечує абсолютну симетрію вузлових точок хвилі без використання дробів.
* **Нульовий фазовий джитер:** Керування частотами $f_1$ та $f_2$ відбувається виключно додаванням цілого "кроку приросту" до акумулятора фази під час кожного такту таймера.
* **Безперервність фази:** Зміна частоти відбувається миттєво, без розривів фази (glitches) та розсинхронізації між двома електромагнітними котушками.

## ⚙️ Апаратна архітектура

Прошивка оптимізована для двоядерних мікроконтролерів **ESP32-S3** (Bare-metal / FreeRTOS).

Процеси фізично розділені між ядрами процесора для забезпечення атомарної точності:
* **Core 0 (DDS Engine):** Виділено виключно під цілочисельну математику акумулятора фази та утримання таймінгів. Зчитує значення з попередньо згенерованої таблиці (Look-Up Table, LUT) та генерує ШІМ-сигнали (PWM) для двох котушок індуктивності.
* **Core 1 (UI & Control):** Обробляє асинхронні переривання від фізичних енкодерів, дозволяючи оператору прецизійно налаштовувати частоту носіїв та глибину інтерференції ($\Delta f$) у реальному часі без впливу на генерацію хвиль.

## 🛠 Обладнання для прототипування

Для збірки фізичного стенду знадобляться:
1. Мікроконтролер ESP32-S3 (наприклад, DevKitC).
2. Два модулі підсилення струму (наприклад, MOSFET-драйвери або H-мости), сумісні з частотами ШІМ.
3. Дві електромагнітні котушки (індуктивність розраховується під цільовий частотний діапазон).
4. Роторні енкодери для ручного керування частотами $f_1$ та $f_2$.
